rasterio.shutil module¶

Raster management.

rasterio.shutil.copy()¶

Copy a raster from a path or open dataset handle to a new destination with driver specific creation options.

Parameters:

src : str or dataset object opened in ‘r’ mode

Source dataset

dst : str

Output dataset path

driver : str, optional

Output driver name

strict : bool, optional. Default: True

Indicates if the output must be strictly equivalent or if the driver may adapt as necessary

creation_options : **kwargs, optional

Creation options for output dataset
rasterio.shutil.copyfiles()¶

Copy files associated with a dataset from one location to another.

Parameters:

src : str

Source dataset

dst : str

Target dataset
rasterio.shutil.delete()¶

Delete a GDAL dataset

Parameters:

path : path

Path to dataset to delete

driver : str or None, optional

Name of driver to use for deleting. Defaults to whatever GDAL determines is the appropriate driver
rasterio.shutil.exists()¶

Determine if a dataset exists by attempting to open it.

Parameters:

path : str

Path to dataset
